# 🏅 인정협회

> **당신의 주장, 인정? 노인정?**

<br />

## 인정협회가 뭐야?

주장을 올리면 다른 사람들이 **인정** 또는 **노인정**으로 투표해주는 커뮤니티 서비스야.
맞으면 인정, 틀리면 노인정. 그게 다야.

<br />

## 어떻게 써?

1. **로그인** — 카카오 또는 구글로 간단하게
2. **주장 올리기** — 하고 싶은 말 써서 올려봐
3. **인정 or 노인정** — 다른 사람 주장에 투표해줘
4. **인정률 확인** — 내 주장이 얼마나 인정받았는지 확인해봐
5. **공유하기** — 재밌는 주장 친구한테 공유해봐

<br />

## 기능

| 기능 | 설명 |
|------|------|
| 🗳️ 인정 / 노인정 투표 | 주장마다 실시간 투표 |
| 📊 인정률 통계 | 인정률을 퍼센트로 확인 |
| 🔥 정렬 필터 | 최신순 / 인기순 / 인정률높은순 / 치열한순 |
| 💬 댓글 & 대댓글 | 주장에 댓글 달고 답글 달기 |
| ✏️ 수정 / 삭제 | 내 글, 댓글 수정 및 삭제 |
| 🚨 신고 | 글, 댓글, 프로필 신고 |
| 👤 프로필 | 닉네임 + 이모지로 나만의 프로필 |
| 🔗 공유 | 네이티브 공유 또는 링크 복사 |
| 📱 PWA | 홈 화면에 추가해서 앱처럼 사용 |

<br />
