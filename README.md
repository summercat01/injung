# 인정협회

> 당신의 주장, 인정? 노인정?

주장을 올리고 다른 사람들의 인정을 받아보세요.
인정이면 인정, 아니면 노인정.

## 기술 스택

- **Framework**: Next.js 15 (App Router)
- **Auth**: NextAuth.js v5 (Kakao, Google)
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS v4

## 시작하기

```bash
npm install
npm run dev
```

환경변수 설정 (`.env.local`):

```
DATABASE_URL=
AUTH_SECRET=
AUTH_KAKAO_ID=
AUTH_KAKAO_SECRET=
AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
```
